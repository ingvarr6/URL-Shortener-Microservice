URL Shortener Microservice

Example creation usage:

https://ing6-urlshortener.glitch.me/new/https://www.google.com
https://ing6-urlshortener.glitch.me/new/http://foo.com:80

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"https://ing6-urlshortener.glitch.me/1111" }

Usage:

https://ing6-urlshortener.glitch.me/1111

Will redirect to:

https://www.google.com/
